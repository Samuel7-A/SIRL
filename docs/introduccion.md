# Introducción

## Contexto

En el mundo actual, la cantidad de libros disponibles crece de forma exponencial.
Encontrar una lectura adecuada según los gustos personales se convierte en una tarea
difícil para muchos usuarios. Los sistemas de recomendación inteligentes ofrecen una
solución eficaz a este problema.

## Problema

Los usuarios no siempre saben qué libro elegir. Necesitan una herramienta que,
a partir de sus preferencias (género, autor favorito, temática), sea capaz de
sugerirles libros de forma automática y razonada.

## Propuesta

El presente proyecto propone el desarrollo de **SIRL** (Sistema Inteligente de
Recomendación de Libros), un sistema multilenguaje que integra tres paradigmas
de programación distintos:

- **Scala** para la lógica principal y coordinación del sistema.
- **Prolog** para el razonamiento lógico y la base de conocimiento.
- **Python** como lenguaje de integración y procesamiento de datos.

Esta arquitectura permite aprovechar las fortalezas específicas de cada tecnología,
promoviendo modularidad, extensibilidad y eficiencia.
